# 用户使用手册

## 欢迎使用"吃什么"餐饮聚合推荐平台！

### 平台简介

"吃什么"是一个智能餐饮推荐平台，聚合了美团、大众点评、小红书等多个平台的数据，通过AI算法为您推荐最合适的美食。

---

## 快速开始

### 1. 首页搜索

#### 智能搜索
在首页搜索框输入您的口味偏好，例如：
- "想吃辣的火锅"
- "清淡的日料"
- "便宜的快餐"

AI会自动理解您的意图并推荐餐厅。

#### 热门标签
点击热门标签快速筛选：
- 🔥 辣味
- 🍜 清淡
- 🍰 甜品
- 🍣 日料
- 🥘 火锅

### 2. 今日推荐

每天为您精选3家餐厅，基于：
- 您的口味偏好
- 当前天气
- 季节特点
- 热门趋势

## 核心功能

### 智能搜索

#### 口味搜索
支持自然语言输入：
- "想吃麻辣的川菜"
- "附近有什么好吃的"
- "适合约会的餐厅"

#### 筛选条件
- **价格区间**: ¥ / ¥¥ / ¥¥¥ / ¥¥¥¥
- **距离范围**: 500m / 1km / 3km / 5km
- **评分门槛**: 3.0+ / 4.0+ / 4.5+
- **菜系**: 川菜/粤菜/日料/韩餐等
- **营业状态**: 营业中/已打烊

### 餐厅详情

#### 多平台评分对比
查看美团、大众点评、小红书的评分对比：
- ⭐⭐⭐⭐⭐ 美团: 4.8分
- ⭐⭐⭐⭐⭐ 点评: 4.6分
- ❤️ 98% 小红书推荐

#### AI评论摘要
自动聚合多平台评论，生成一句话总结：
> "味道正宗，分量足，服务好，环境嘈杂"

### 智能推荐

#### 天气推荐
根据天气自动推荐：
- ☀️ 夏天 → 凉皮、冰粉、沙拉
- ❄️ 冬天 → 火锅、烤肉、热汤
- 🌧️ 雨天 → 麻辣烫、粥品

#### 心情推荐
根据心情选择：
- 😄 开心 → 甜品、烤肉
- 😢 难过 → 巧克力、炸鸡
- 😴 疲惫 → 清淡汤品
- 🎉 庆祝 → 高级日料、西餐

#### 随机推荐
选择困难？摇一摇随机推荐！

### 口味画像

系统会学习您的偏好：
- 分析搜索关键词
- 统计收藏菜系
- 记录打卡历史
- 生成口味标签（如"辣王"、"日料控"）

#### 查看画像
雷达图展示您的口味维度：
- 🌶️ 辣度
- 🍬 甜度
- 🍋 酸度
- 🧂 咸度
- 🐟 鲜度
- 🍃 清淡度

## 收藏与打卡

### 收藏夹

#### 添加收藏
1. 在餐厅详情页点击"收藏"
2. 选择分组：
   - 💚 想去
   - ✅ 已去
   - ⭐ 收藏
   - 📁 自定义分组

#### 管理收藏
- 拖拽移动分组
- 批量删除
- 导出数据

### 美食打卡

#### 记录打卡
1. 到店后点击"打卡"
2. 上传照片
3. 写评价
4. 打分

#### 地图展示
在地图上查看：
- 🟢 已打卡
- 🔴 想去
- 🔵 附近推荐

## 常见问题

### Q: 搜索结果不准确怎么办？
A: 
1. 尝试更具体的描述
2. 使用筛选条件
3. 系统会学习您的偏好

### Q: 为什么某些餐厅没有评分？
A: 
1. 新开餐厅数据未更新
2. 可以手动启动爬虫更新
3. 管理员 → 爬虫管理 → 启动爬取

### Q: 如何修改口味画像？
A: 
1. 个人中心 → 口味画像
2. 手动调整偏好
3. 清空历史重新学习

### Q: 收藏夹有数量限制吗？
A: 
无限制，但建议定期整理。

### Q: 爬虫多久更新一次数据？
A: 
- 自动：每周更新热门餐厅
- 手动：可随时启动爬虫
- 后台：管理员配置定时任务
